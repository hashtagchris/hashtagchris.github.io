For 2025, I'm attempting to use my main GitHub pages site as a Bluesky feed generator.

solomon-threads feed on Bluesky

I used this script to register the feed with Bluesky.

paths

  • .well-known/did.json
  • xrpc/app.bsky.feed.getFeedSkeleton with the right content-type via index.json

credits

Dave Peck's blog helped me figure out I needed a file under `.well-known`
